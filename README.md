## Crimson Midnight

Crimson Midnight is an AI-powered murder mystery investigation game where players take on the role of a detective. 
Analyze cases, interrogate suspects, and uncover the truth before time runs out.

## About the Game

Crimson Midnight is an interactive murder mystery investigation game where players act as detectives.

The player must
* Study the case file
* Interrogate each suspect
* Take notes and analyze responses
* Decide who the killer is

At the end, the game reveals whether the player made the correct choice.

## Key Features
* Immersive UI
* Multiple suspects with dynamic responses
* AI-Powered Interrogation
* Note-Taking System
* Story-based gameplay
* Multiple Story Scenario
* Decision-Based Ending
* Replayable with different strategies

## Game Flow

* Main Page
* Menu (Select Story)
* Case File (Read Story)
* Choose Suspect
* Interrogation Room
* Repeat for all suspects
* Final Decision
* Ending (Success / Fail)
